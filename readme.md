# cafemates-services
Services Cafemates

# Getting Started
clone poject
`git clone https://github.com/telecreative-developer/cafemates-services/`

# Requirements

* node v8.11.1 
* npm 5.6.0
* phyton
* postgresql


# Installing
`cd cafemates-services`
`bash package.sh`
`import database dbexport.pgsql`

# Configuration 
`create file .env, copy script from .env.example`

`

DATABASE_HOST =  host
